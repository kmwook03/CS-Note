## 컴퓨터의 구성 요소

> **중앙처리장치(CPU; Central Processing Unit)**

메모리에 저장된 명령어를 읽고, 해석하고, 실행하는 컴퓨터의 두뇌.
내부에 산술논리연산장치(ALU; Arithmetic Logic Unit), 레지스터(Register), 제어장치(CU; Control Unit)가 존재한다.
- ALU: 컴퓨터 내부에서 수행되는 연산 처리
- 레지스터: CPU 내부의 임시 저장 장치
- CU: 제어 신호(control signal)라는 전기 신호를 전송하고 명령어를 해석하는 장치

> **주기억장치(Main Memory)**

현재 실행되는 프로그램의 명령어와 데이터를 저장하는 부품.
저장된 값에 빠르고 효율적으로 접근하기 위해 주소(address) 개념이 사용된다.
- RAM(Random Access Memory)
휘발성 메모리로 전원이 꺼지면 저장된 데이터가 사라진다.
읽기와 쓰기가 빨라서 주기억장치로 사용된다.
- ROM(Read Only Memory)
비휘발성 메모리로 전원이 꺼져도 데이터가 유지된다.
읽기는 빠르지만 쓰기는 불가능하거나 느리다.
주로 보조기억장치로 사용되나 컴퓨터 부팅 시에 필요한 BIOS 등은 ROM에 저장되기 때문에 주기억장치 범주에 포함될 수 있다.

> **보조기억장치(Secondary Storage)**

주기억장치보다 큰 비휘발성 메모리.

> **입출력장치(Input/Output(I/O) Device)**

컴퓨터 외부에 연결되어 컴퓨터 내부와 정보를 교환하는 장치.

> **메인보드(Main Board/Mother Board)**

CPU, 주기억장치 등 핵심 부품들을 부착할 수 있는 슬롯과 연결 단자를 가진 보드.
메인보드에 연결된 부품들은 서로 정보를 주고받을 수 있다.

> **시스템 버스(System Bus)**

부품들이 서로 정보를 주고받는 데이터 통로.
- 주소 버스(Address Bus): 주소를 주고받는 통로. 단방향.
- 데이터 버스(Data Bus): 명령어와 데이터를 주고받는 통로. 양방향.
- 제어 버스(Control Bus): 제어 신호를 주고받는 통로. 양방향.
