### 1. 신호와 주파수의 기초 (Fundamentals of Signal and Frequency)

**핵심 용어 (Terminology)**
*   **Analog (연속형, Continuous):** 시간에 대해 무한한 해상도와 연속적인 진폭을 가지는 신호입니다 (예: 음성, 온도).
*   **Digital (이산형, Discrete):** 불연속적인 시간 단계에서만 정의되며, 진폭이 유한한 레벨로 양자화(Quantized)된 0과 1의 이진 데이터입니다.
*   **Sampling (표본화):** 아날로그 신호를 컴퓨터에서 처리할 수 있도록 디지털 데이터로 변환하는 과정입니다 (예: 8000Hz 샘플링 레이트).
*   **Amplitude ($A$, 진폭):** 신호의 최대 크기로, 파형의 수직적 크기를 결정합니다.
*   **Frequency ($B$ or $f$, 주파수):** 신호가 얼마나 빨리 진동하는지를 나타내며, 주기의 역수($1/T$)입니다.
*   **Phase ($C$, 위상):** 파형의 수평적 이동(시작 위치)을 결정합니다.
*   **Time Domain ($x(t)$, 시간 도메인):** 시간에 따라 신호가 어떻게 변화하는지 보여주는 관점입니다.
*   **Frequency Domain ($X(f)$, 주파수 도메인):** 신호가 어떤 주파수 성분들로 구성되어 있는지(스펙트럼 구성)를 보여주는 관점입니다.
*   **Spectrogram (스펙트로그램):** 신호의 주파수 스펙트럼이 **시간에 따라 어떻게 변화하는지** 시각적으로 보여주는 도구입니다.

**기술의 적용법 (Applications)**
*   **복합 신호의 분해 및 식별:** '도, 미, 솔'이 합쳐진 복잡한 화음 파형(Time Domain)을 주파수 도메인으로 변환(FFT)하면, 각각 440Hz(도), 557Hz(미) 등의 독립된 주파수 성분으로 분리하여 쉽게 식별할 수 있습니다.
*   **DTMF (Dual-Tone Multi Frequency) Signaling:** 현대 전화기 키패드에 적용되는 기술입니다. 예를 들어 숫자 '1'을 누르면 697Hz와 1209Hz 두 개의 특정 주파수가 합쳐진 신호가 발생하며, 전화 교환국은 이 주파수 조합을 분석하여 어떤 버튼이 눌렸는지 식별합니다.

### 2. 푸리에 변환 (Fourier Transform)

복잡한 신호를 다루기 쉬운 단순한 기본 주기 함수의 합으로 분해하는 수학적 기법입니다.

**핵심 용어 (Terminology)**
*   **Linearity (선형성):** 복잡한 현상이나 함수를 관리하기 쉬운 단순한 요소들의 선형 결합(Linear Combination)으로 쪼개어 표현하는 성질입니다.
*   **Euler’s Formula (오일러 공식):** $e^{ix} = \cos x + i \sin x$. 삼각함수를 복소 지수함수로 변환하여, 진폭과 위상(Magnitude & Phase)을 하나의 복소수 수식으로 통합 표현하게 해줍니다.
*   **Phasor (페이저):** 신호를 복소 평면 위에서 회전하는 벡터로 취급하는 개념으로, QAM 같은 변조 기법 이해에 필수적입니다.
*   **Fourier Series (푸리에 급수):** **주기 신호(Periodic signal)**를 기본 주파수($f_0$)의 배수(Harmonics)를 가지는 복소 지수 함수의 선형 결합으로 표현합니다.
*   **Sinc Function (싱크 함수):** $\text{sinc}(x) = \frac{\sin(\pi x)}{\pi x}$. 구형파(Square Wave)의 푸리에 계수 포락선(Envelope)을 그릴 때 나타나는 함수입니다.
*   **Bandwidth (대역폭):**
    *   *Spectrum:* 신호에 포함된 주파수 범위.
    *   *Absolute bandwidth:* 스펙트럼의 전체 너비.
    *   *Effective bandwidth:* 신호 에너지의 대부분을 포함하는 좁은 주파수 대역.
*   **DC Component (직류 성분):** 주파수가 0인 성분($f=0$)으로, 신호의 '평균값(Average Value)' 또는 '편향(Offset)'을 의미합니다.

**기술의 적용법 (Applications)**
*   **4가지 푸리에 변환 쌍의 적용:** 신호가 연속적인지 이산적인지, 주기적인지 비주기적인지에 따라 4가지(CTFS, DTFS/DFT, CTFT, DTFT) 변환 기법을 골라 적용합니다.
    *   비주기적 신호(Aperiodic Signal)는 주기가 무한대($P \rightarrow \infty$)인 주기 신호의 극한으로 간주하여 푸리에 변환(Transform)을 적용합니다.
*   **신호 왜곡(Signal Distortion) 분석:** 원래의 통신 신호를 전송할 때 무한한 주파수를 다 보낼 수 없으므로 제한된 대역폭(Limited Bandwidth, $N$번째 차수까지만 취함)을 사용하게 되며, 이로 인해 필연적으로 발생하는 신호 왜곡 현상을 분석하는 데 사용됩니다.
*   **연산의 단순화:** 오일러 공식을 활용해 복잡한 삼각함수 연산을 지수 법칙($e^A \cdot e^B = e^{A+B}$)을 이용한 단순한 산술 연산으로 바꾸어, 데이터 통신에서의 믹싱, 필터링, 변조 과정을 극적으로 단순화합니다.

### 3. 선형 시불변 시스템 (LTI System)

입력 신호가 통신 채널(시스템)을 통과할 때 어떻게 변화하는지 분석하기 위한 시스템 모델입니다.

**핵심 용어 (Terminology)**
*   **LTI System (Linear Time-Invariant System, 선형 시불변 시스템):** 선형성(입력의 크기에 비례하여 출력이 커지고 더해짐)과 시불변성(입력이 지연되면 출력도 동일하게 지연됨)을 모두 만족하는 시스템입니다.
*   **Causal System (인과적 시스템):** 현재의 출력 $y(t_0)$가 과거와 현재의 입력값에만 의존하고, 미래의 입력값에는 의존하지 않는 현실적인 시스템입니다.
*   **Impulse (임펄스, $d(t)$ 또는 Dirac Delta $\delta(t)$):** $t=0$에서만 무한대의 크기를 가지고 나머지 시간에서는 0이며, 적분 면적이 1인 이상적인 신호입니다.
*   **Impulse Response ($h(t)$, 임펄스 응답):** 시스템에 임펄스 신호를 입력했을 때 나오는 시스템의 고유한 출력(응답)입니다.
*   **Convolution Integral (컨볼루션/합성곱 적분, $*$):** 임의의 입력 신호 $x(t)$와 시스템의 임펄스 응답 $h(t)$를 곱하고 적분하여, 시스템의 최종 출력 $y(t)$를 구하는 수학적 연산입니다 ($y(t) = x(t) * h(t)$).
*   **Eigenfunctions (고유함수):** LTI 시스템을 통과해도 '모양(Shape)'이 변하지 않고 진폭과 위상만(복소 상수 곱) 변하는 함수입니다. 복소 지수함수($e^{st}$ 또는 $e^{i\omega t}$)가 대표적입니다.

**기술의 적용법 (Applications)**
*   **시스템 특성 파악 및 출력 예측 (Impulse Response & Convolution):**
    *   어떤 공간(예: 콘서트홀)이나 통신 채널의 특성을 파악하기 위해 박수 소리 같은 임펄스($d(t)$)를 발생시키고, 메아리 패턴인 임펄스 응답($h(t)$)을 측정하여 시스템의 '지문'을 알아냅니다.
    *   이후 오케스트라 연주 같은 복잡한 임의의 입력 신호 $x(t)$가 들어오면, 이 입력 신호를 무한히 많은 임펄스들의 합으로 분해한 뒤, 앞서 구한 임펄스 응답 $h(t)$와 컨볼루션 연산($*$)을 하여 최종 출력 소리 $y(t)$를 정확히 예측할 수 있습니다.
*   **고유함수(Eigenfunctions)를 이용한 연산 최적화 (The Big Picture):**
    *   시간 도메인에서의 컨볼루션 연산($x(t) * h(t)$)은 계산이 매우 복잡하고 어렵습니다.
    *   그러나 푸리에 변환의 기본 주기 함수인 복소 지수함수 $e^{st}$ (Eigenfunction)를 입력으로 사용하면, 시간 도메인에서의 복잡한 컨볼루션 연산이 주파수 도메인에서는 단순한 **곱셈 연산($H(s) \cdot e^{st}$)**으로 바뀝니다. 따라서 모든 입력을 복소 지수함수의 선형 결합으로 바꾸는 푸리에 변환이 데이터 통신 시스템 분석에 필수적으로 적용됩니다.
